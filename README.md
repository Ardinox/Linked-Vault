# LinkedVault

A robust, WEB-based Employee Management System built in C.

### How to Run Frontend

1. Open `frontend/landing.html` (The entry point).

2. start live server.

### How to Run Backend

1. Open your terminal in `backend` folder.

2. Compile the project:

   - **Windows:**
     ```bash
     mingw32-make
     ```
   - **Linux / Mac:**
     ```bash
     make
     ```

3. Start the server:
   - **Windows:**
     ```bash
     .\LinkedVault.exe
     ```
   - **Linux / Mac:**
     ```bash
     ./LinkedVault.exe
     ```

---

**Server Address:** `http://localhost:8000`
